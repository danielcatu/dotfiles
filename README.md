# Setup
Clone into `~/.config/dotfiles`
Run `./setup.sh`

## iTerm2 profile
- Import `iterm2/sjdonado_iterm2_profile.json`
- Custom settings: Scroll wheel sends arrow keys when in alternate `screen mode: true`

## Config nvim plugins
```vim
:PackerSync
:MasonToolsInstall
:Copilot auth
```
